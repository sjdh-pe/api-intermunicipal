PE Livre Acesso Intermunicipal — API

Bem-vindo! Esta aplicação expõe uma API REST para gestão do benefício de transporte intermunicipal em Pernambuco, incluindo cadastro e consulta de beneficiários, responsáveis, uploads de documentos e lookups de referência.

Documentação

A melhor forma de explorar os endpoints é pela interface interativa do Swagger.

Abrir Swagger UI OpenAPI (JSON)

Como começar (dev)

  • Defina a variável de ambiente APP_PROFILE=dev para liberar as rotas em ambiente local.
  • Inicie a aplicação: ./mvnw spring-boot:run. Em seguida, acesse /swagger.

CORS e outras opções podem ser ajustadas via variáveis de ambiente. Consulte o README para detalhes.
